* each 메서드

        배열의 모든 인덱스에 순차적으로 접근하고자 할 때, 객체가 가지고 있는
        모든 속성에 순차적으로 접근하고자 할 때 사용하는 메서드 (for of, for in 의
        기능이 모두 들어가 있다)

        [표현법]
        1)  $.each(객체/배열, function(매개변수1, 매개변수2) {
                실행할 내용
                객체 : 속성, 속성값 / 배열 : 인덱스, 값
            });
        2)  $("선택자"/배열).each(function(매개변수1, 매개변수2) {
                실행할 내용
            });
